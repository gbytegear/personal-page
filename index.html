BinOM(Binary object model) - database and format of serialization/deserialization of data implemnted as C++ library

BinOM Data Types

  • byte - 8 bit variable
  • word - 16 bit variable
  • dword - 32 bit variable
  • qword - 64 bit variable
  • byte array - 8 bit values array(can be interpreted as UTF-8 string)
  • word array - 16 bit values array(can be interpreted as UTF-16 string)
  • dword array - 32 bit values array
  • qword array - 64 bit values array
  • array - heterogeneous array
  • object - heterogeneous key-value container
